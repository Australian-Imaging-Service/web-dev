# Vectra Export Workflows

# Summary

Cleansed Vectra exports happen via the VectraDBTool.exe utility. Exports can be initiated by the GUI or the command-line. There is a specific set of command-line arguments that produces the same exported files as the GUI. In either case, there is a requirement to supply/select a list of study IDs to be exported. There are several options for producing this list, however they have varying impacts on technical or staff resources.

# VectraDBTool
# GUI

Version 1.8 (and newer) of the VectraDBTool.exe utility can export cleansed patient data via the GUI (Tools > Export Patients). This requires the user to manually select the list of patients to export from the master list, and also requires the user to tick several checkboxes (cleanse patient data, and backup images) if they are unticked.
